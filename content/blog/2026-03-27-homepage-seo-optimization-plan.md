# Homepage SEO Optimization Plan

**Current Status:** Position 24, CTR 4.63%, 971 impressions, 45 clicks (90 days)  
**Target:** Position 10, CTR 8%+, estimated +30 clicks/month  
**Priority:** 🔴 Urgent (Complete by 2026-04-10)

---

## Problem Analysis

### Current Homepage Performance

| Metric | Value | Industry Benchmark | Gap |
|--------|-------|-------------------|-----|
| **Average Position** | 24.3 | Top 10 | -14 positions |
| **CTR** | 4.63% | 8-12% (position 24) | -3-7% |
| **Impressions** | 971 | - | Good visibility |
| **Clicks** | 45 | - | Underperforming |

### Root Causes

1. **Title Tag Not Optimized**
   - Current: Likely generic or brand-only
   - Missing: Core keywords (AI Glasses, Smartwatch OEM, B2B)
   - Missing: Value propositions (MOQ, certifications, lead time)

2. **Meta Description Weak**
   - Not compelling enough for B2B buyers
   - Missing: Clear CTAs, unique selling points

3. **Content Thin**
   - Homepage likely lacks depth for target keywords
   - Missing: Product categories, certifications, trust signals

4. **Internal Linking Poor**
   - AI Glasses page (61% of traffic) not linking back to homepage effectively
   - Missing: Contextual links from high-performing pages

5. **Technical SEO Issues**
   - Page load speed may be suboptimal
   - Missing: Structured data (Organization, Product schema)
   - Missing: Optimized images with alt text

## Optimization Strategy

### Phase 1: On-Page Elements (Days 1-3)

#### 1.1 Title Tag Optimization

**Current (assumed):** `Microwear - Smart Watch Manufacturer`

**New Title:**
```
AI Smart Glasses & Smartwatch OEM Manufacturer | Microwear B2B
```

**Rationale:**
- Front-loads high-value keyword "AI Smart Glasses" (position 3.93, 36.9% CTR)
- Includes "Smartwatch OEM" (365 impressions, position 55 - opportunity)
- Adds "B2B" to signal commercial intent
- Keeps brand name at end (already recognized)
- Length: 62 characters (optimal for Google)

#### 1.2 Meta Description Optimization

**New Meta Description:**
```
Professional AI glasses & smartwatch OEM/ODM manufacturer. MOQ 100pcs, CE/FCC/RoHS certified, 15+ years experience. Get quote in 24h. Free samples for B2B buyers.
```

**Rationale:**
- Includes primary keywords (AI glasses, smartwatch OEM/ODM)
- Highlights key B2B decision factors:
  - MOQ (100pcs - low barrier)
  - Certifications (CE/FCC/RoHS - trust signals)
  - Experience (15+ years - credibility)
  - Response time (24h - service quality)
  - Free samples (conversion incentive)
- Clear CTA ("Get quote")
- Length: 158 characters (optimal)

#### 1.3 H1 Tag Optimization

**Current (assumed):** `Welcome to Microwear` or similar

**New H1:**
```
Your Trusted AI Glasses & Smartwatch OEM Partner
```

**Rationale:**
- Includes both product categories
- Emphasizes partnership (B2B relationship)
- Uses "Trusted" as trust signal

### Phase 3: Technical SEO (Days 8-10)

#### 3.1 Structured Data

**Add JSON-LD to Homepage:**
```html
<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Organization",
  "name": "Microwear",
  "url": "https://microwear.info",
  "logo": "https://microwear.info/logo.png",
  "description": "Professional AI glasses and smartwatch OEM/ODM manufacturer with 15+ years experience",
  "foundingDate": "2010",
  "address": {
    "@type": "PostalAddress",
    "addressCountry": "CN"
  },
  "contactPoint": {
    "@type": "ContactPoint",
    "contactType": "sales",
    "availableLanguage": ["English", "Chinese", "German"]
  },
  "sameAs": [
    "https://www.linkedin.com/company/microwear",
    "https://twitter.com/microwear"
  ]
}
</script>

<script type="application/ld+json">
{
  "@context": "https://schema.org",
  "@type": "Product",
  "name": "AI Smart Glasses",
  "description": "Micro-LED display smart glasses with Android XR integration",
  "brand": {
    "@type": "Brand",
    "name": "Microwear"
  },
  "offers": {
    "@type": "Offer",
    "availability": "https://schema.org/InStock",
    "minOrderQuantity": 100,
    "businessFunction": "wholesale"
  }
}
</script>
```

#### 3.2 Image Optimization

**Audit All Homepage Images:**
- [ ] Add descriptive alt text (include keywords)
- [ ] Compress images (WebP format, <100KB each)
- [ ] Implement lazy loading
- [ ] Add width/height attributes (prevent CLS)

**Example:**
```html
<img 
  src="/images/hero/ai-glasses-hero.webp" 
  alt="AI Smart Glasses OEM Manufacturing - Microwear Factory"
  width="1200"
  height="630"
  loading="eager"
/>
```

#### 3.3 Page Speed Optimization

**Target Metrics:**
- LCP (Largest Contentful Paint): <2.5s
- FID (First Input Delay): <100ms
- CLS (Cumulative Layout Shift): <0.1

**Actions:**
- [ ] Enable gzip/brotli compression
- [ ] Minify CSS/JS
- [ ] Implement CDN
- [ ] Preload critical resources
- [ ] Defer non-critical JS

## Risk Mitigation

### Risk 1: Position Drops After Update
**Mitigation:**
- Keep backup of original content
- Monitor daily for first 2 weeks
- Ready to revert if position drops >5 positions

### Risk 2: No Improvement After 4 Weeks
**Mitigation:**
- Conduct competitor analysis
- A/B test different title/description combinations
- Consider additional content additions

### Risk 3: Technical Issues
**Mitigation:**
- Test on staging environment first
- Backup before deployment
- Have rollback plan ready
