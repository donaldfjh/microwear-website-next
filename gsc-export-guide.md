# Google Search Console 数据导出指南

## 📊 每周 SEO 监控流程

### 步骤 1：访问 Google Search Console

打开：https://search.google.com/search-console

选择你的网站：`microwear.info`

---

### 步骤 2：进入效果报告

1. 左侧菜单点击 **"效果"** (Performance)
2. 确保时间范围选择 **"最近 3 个月"** 或 **"最近 28 天"**
3. 勾选以下指标：
   - ✅ 总点击次数
   - ✅ 总展示次数
   - ✅ 平均点击率
   - ✅ 平均排名

---

### 步骤 3：筛选 AI Glasses 相关关键词

1. 点击 **"+ 新建"** 筛选器
2. 选择 **"查询"** (Query)
3. 选择 **"包含"** (Containing)
4. 输入以下关键词（每次筛选一个）：

**第一批：AI Glasses**
```
glasses
```

**第二批：智能眼镜（中文）**
```
眼镜
```

**第三批：Smart Watch**
```
watch
```

---

### 步骤 4：导出数据

1. 筛选后，点击表格上方的 **下载图标** ⬇️
2. 选择 **"Google 表格"** 或 **"CSV"**
3. 保存文件，命名格式：
   ```
   gsc-export-YYYY-MM-DD-glasses.csv
   ```

---

### 步骤 5：发送给我分析

把导出的 CSV 文件内容复制粘贴给我，或者告诉我文件路径。

我会分析：
- 哪些关键词排名上升/下降
- 哪些有机会优化到第一页
- 需要调整哪些页面内容

---

## 📋 建议导出时间

**频率：** 每周一次
**建议时间：** 周一上午（数据包含完整一周）

**示例时间表：**
- 第 1 周周一：导出并分析
- 第 2 周周一：导出并分析
- 对比两周数据，看趋势

---

## 🎯 重点关注的指标

| 指标 | 含义 | 优化目标 |
|------|------|---------|
| **展示次数** | Google 展示你网站的次数 | 增加 |
| **点击次数** | 用户点击你网站的次数 | 增加 |
| **点击率 (CTR)** | 点击次数 ÷ 展示次数 | > 3% |
| **平均排名** | 搜索结果中的平均位置 | < 10（第一页） |

---

## ❓ 常见问题

### Q: 为什么有些关键词没有数据？
A: 可能是因为展示次数太少，Google 不显示具体数据（显示为 0 或 -）

### Q: 平均排名多少算好？
A: 
- 1-3 名：优秀（前 3 结果）
- 4-10 名：良好（第一页）
- 11-20 名：有优化空间（第二页）
- 20+ 名：需要重点优化

### Q: 点击率多少算正常？
A: 
- 第一名：约 30% CTR
- 第五名：约 5% CTR
- 第十名：约 2% CTR
